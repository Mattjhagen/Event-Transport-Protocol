# ETP SDK (Draft)

This directory contains experimental client libraries.

Currently, the ETP standard is maintained as a set of TypeScript types and Zod schemas available in `src/types.ts`.

Future plans include:
- `@etp/core`: Lightweight parser and validator.
- `@etp/react`: Hooks for real-time event synchronization.
- `@etp/node`: Server-side utilities for routing and signing.
